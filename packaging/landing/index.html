Deploy your own stack
on your own hardware

LocalSURV is a self-hosted deploy platform — GitHub push-to-deploy, live build logs, reverse proxy, SSL, Cloudflare Tunnel, and managed databases, all from a single binary on a Mac mini or Linux box you already own.

View on GitHub → 5-minute quickstart 
$ curl -fsSL https://localsurv.dev/install.sh | bash

What's in the box

🚀

Git push-to-deploy

GitHub PAT or SSH key, auto-detected Node/Python/Docker builds, live-streamed build logs, webhooks, or 60 s GitOps polling.

🌐

Host-based reverse proxy

Map any domain to a local port. Automatic Let's Encrypt (HTTP-01 or DNS-01) or Cloudflare Tunnel integration — pick your poison.

📊

Observability that ships

Per-service CPU/memory, live WebSocket logs, deployment progress, a system health score, and Discord/Slack notifications.

🗄️

Managed databases

One-click Postgres, MySQL, Redis, Mongo. Auto-inject DATABASE_URL, one-click pg_dump backups, SQL seed runner.

🔒

Encrypted at rest

AES-256-GCM on every secret: env vars, GitHub PATs, Cloudflare tokens. Never logged, never echoed.

🧱

Zero daemons

Single Node process, single SQLite file. No Redis, no Postgres for state, no external queue.
